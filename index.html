  • Host de repositorios GIT
  • Es un sistema de creación de repositorios y control de versiones.
  • Una red social enfocada al desarrollo de código

Git es un sistema de control de versiones de código abierto y gratuito, diseñado para manejar desde proyectos pequeños a proyectos grandes.

  • Compartir cualquier tipo de código
  • Creación de Repositorios
  • Evolución del código
  • Comunicación entre desarrolladores

Es un espacio en la nube donde se almacenan archivos, proyectos y código.

Público

    Ventajas

  • Publicación gratuita.
  • Acceso a código de terceros.

Privado

    Ventajas

  • Protección de derechos de autor e información confidencial.
